# System Status Script

A mini Bash CLI tool to display system information with colors.

## Features

- Shows RAM: Total, Used, Available
- Shows Disk: Total, Used, Available
- Shows Uptime
- Shows Users: names and count
- Color-coded sections for better readability in terminal

## Usage

1. Open terminal
2. Navigate to the folder containing `system_report.sh`
3. Run the script:
   ```bash
   bash system_report.sh
